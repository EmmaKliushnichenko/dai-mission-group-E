# Gender Differences in Executive Roles

This repository contains the project files for the analysis of gender differences in access to executive and board-level roles using BoardEx Europe data.

## Project data

The project data and related files are available through the following Sciebo link:

https://tu-dortmund.sciebo.de/s/wY5kFY6CkMk9LPr

## Reproducibility

To reproduce the analysis, install the required Python packages from `requirements.txt`:

```bash
pip install -r requirements.txt
```

## Project structure

The project includes data preparation, causal inference, supervised learning, unsupervised learning, and synthesis of findings.
